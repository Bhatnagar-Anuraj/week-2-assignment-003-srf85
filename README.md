# DIGM 131 - Scripting for Digital Media

Course materials released week by week.

## Getting new content each week

```bash
git pull origin main
```

## Running self-check tests

Before submitting an assignment, run the self-check:

```bash
cd assignment && python test_assignment.py
```

## Structure

- `demos/` - Code shown in class. Study these!
- `assignments/` - Your assignment skeletons and self-check tests
